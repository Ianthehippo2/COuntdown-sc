# 🌟 Overview

The **Schooled Official Countdown** is a fully interactive web experience designed to celebrate and track the final days of the Schooled era.

The platform combines:

- ⏳ Real-Time Countdown Timer
- 📺 Live Broadcast Status
- 🎵 Apple Music Styled Now Playing Integration
- 🎆 Final Countdown Firework Tracker
- 📅 Dynamic Daily Schedule System
- ⚡ Thunderstruck Thursday Effects
- ☁️ Animated Cloud System
- 📢 Live Site Notifications
- 📱 Mobile Friendly Design

---

# 🚀 Features

## ⏳ Main Countdown

Displays the exact time remaining until the final day of Schooled.

- Days
- Hours
- Minutes
- Seconds

Automatic launch mode activates during the final minute.

---

## 📅 School Day Tracker

Tracks remaining school days until graduation.

Automatically updates in real time.

---

## 📺 Live Broadcast System

When Schooled goes live:

- LIVE indicator activates
- Watch Live button appears
- Streaming options become available

Supported Platforms:

- Schooled
- Apple TV

---

## 🎵 Music Integration

Apple Music inspired interface displaying:

- Album Art
- Song Title
- Artist Name
- Animated Equalizer

---

## 🎆 Final Countdown Firework Counter

Tracks progress toward the final day using interactive fireworks.

Features:

- Live Updates
- Dynamic Tracking
- Celebration Progress System

---

## 📅 Dynamic Schedule System

Displays:

- Current Letter Day
- Current Class
- Time Remaining
- Full Daily Schedule

Supports:

- Regular School Days
- Final Exam Schedules
- Special Events

---

## ⚡ Thunderstruck Thursday

Every Thursday the website transforms with:

- Dark Storm Theme
- Lightning Effects
- Animated Thunder Background
- Special Thursday Branding

---

## 📢 Notification System

Administrators can broadcast messages site-wide.

Used for:

- Announcements
- Schedule Changes
- Live Events
- Emergency Updates
